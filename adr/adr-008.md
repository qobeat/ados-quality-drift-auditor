## Context

Scores can create false precision and encourage metric substitution. The auditor should diagnose drift rather than operate as a release gate.

## Decision

Metrics:

- measure defined Coordinates;
- use optional 0–10 scores;
- report confidence and evidence status separately;
- may remain unscored;
- do not create automatic build PASS/FAIL.

Final evolution assessment uses improvement, mixed, degradation, unchanged, or insufficient_evidence.

## Alternatives

- Make score thresholds mandatory release gates.
- Avoid Metrics entirely.

## Consequences

- Numeric comparison remains available.
- Semantic rationale remains mandatory.
- Missing evidence cannot be converted into a neutral midpoint.
